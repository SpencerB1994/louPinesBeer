## Overview
<p><i>Lou Pine's Beer</i> is a fictional bar website inspired by the TV series <i>True Blood</i><br>
This project demonstrates my understanding of HTML, CSS, and JavaScript by implementing responsive layouts, light/dark mode functionality, and an interactive discount game.</p>

## Features
<ul>
       <li>Responsive layout</li>
       <li>Light and Dark mode</li>
       <li>Interactive JavaScript discount game</li>
       <li>Custom and stock imagery</li>
</ul>

## Technologies Used
<ul>
       <li>HTML5</li>
       <li>CSS3</li>
       <li>JavaScript</li>
</ul>

## Installation

<p>Clone the repository and open 'index.html' in your browser or click on the githubpage link on the right since it is public.</p>

## Purpose

<p>Demonstrates my understanding of front-end web development through HTML, CSS, and JavaScript.</p>

## What I learned

<p>This project strengthened my understanding of JavaScript event handling, DOM manipulation, responsive layouts, and implementing interactive UI elements.</p>
